# Akemi Design System

## Color Palette

| Usage | Color Name | Hex Code | Visual Vibe |
| :--- | :--- | :--- | :--- |
| **Primary / Hero** | Deep Forest Green | `#1B2F2A` | High-end, botanical, and trustworthy. |
| **Secondary / Accents** | Warm Copper | `#B87333` | Energetic (caffeine) and premium (glow). |
| **Background (Main)** | Creamy Bone | `#F9F5F0` | Soft, clean, and easier on eyes than pure white. |
| **Background (Alt)** | Toasted Almond | `#E2D1C3` | Mimics the burlap texture for section breaks. |
| **Text / Details** | Espresso Roast | `#2D1E17` | Softer than black; feels organic and grounded. |
